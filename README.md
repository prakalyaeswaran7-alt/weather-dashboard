# Weather Dashboard

A simple and responsive Weather Dashboard that displays real-time weather information for any city.

## Features

- Search weather by city name
- View temperature, humidity, and wind speed
- Responsive design
- Easy-to-use interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Weather API
